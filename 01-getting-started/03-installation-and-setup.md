# Installation & Setup

- **Play CDN** for prototyping: `<script src="https://cdn.tailwindcss.com"></script>`
- **CLI**: `npx tailwindcss init`
- **PostCSS plugin**: create `postcss.config.js`
- **Framework‑specific**: Next.js, Vite, Laravel, etc.

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
